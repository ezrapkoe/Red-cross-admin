3How to Change Firebase Account

üîÑStep-by-Step Process

  1. Go to Firebase Console - Visit https://console.firebase.google.com/
  2. Create a new project or select an existing one
  3. Set up authentication - Enable Email/Password sign-in
  4. Create a Realtime Database - Set security rules
  5. Get your configuration - Go to Project Settings ‚Üí General ‚Üí Your apps
  6. Update your JavaScript files - Replace the config values

‚öôÔ∏èWhere to Update Configuration

You need to update the Firebase configuration in all your JavaScript files:

// In index.js, index2.js, index3.js, index4.js, index5.js, login.js, register.js const firebaseConfig = { apiKey: "YOUR_NEW_API_KEY", authDomain: "YOUR_NEW_AUTH_DOMAIN", databaseURL: "YOUR_NEW_DATABASE_URL", projectId: "YOUR_NEW_PROJECT_ID", storageBucket: "YOUR_NEW_STORAGE_BUCKET", messagingSenderId: "YOUR_NEW_SENDER_ID", appId: "YOUR_NEW_APP_ID", measurementId: "YOUR_NEW_MEASUREMENT_ID" };

Important: You must update the configuration in every JavaScript file that uses Firebase. There are 7 files in your project that need updating.

4Setting Up a New Firebase Project

üöÄCreate a New Project

  1. Click "Create a project" in Firebase Console
  2. Name your project (e.g., "kenya-red-cross-admin")
  3. Enable Google Analytics (optional)
  4. Create the project

üîêSet Up Authentication

  1. Go to Authentication ‚Üí Sign-in method
  2. Enable Email/Password provider
  3. Save changes

üóÉÔ∏èSet Up Realtime Database

  1. Go to Realtime Database
  2. Create database - Choose a location
  3. Set security rules - Start in test mode
// Example starter security rules (for development only) { "rules": { ".read": true, ".write": true } }

Security Warning: These rules allow anyone to read and write to your database. For production, implement proper authentication-based rules.

5Get Your New Firebase Config

üìãHow to Find Configuration Values

  1. Go to Project Settings (gear icon)
  2. Select "General" tab
  3. Scroll to "Your apps"
  4. Add a web app if you haven't already
  5. Copy the config values

Tip: You can register multiple web apps in the same Firebase project if you have different environments (development, production).
